# JSON 格式

    {
      title: "福利標題",
      content: "福利內容",
      url: "福利法條網址",
      files: [
        {
          name: "網站上的檔案名稱",
          filename: "實際檔案名稱"
        },
        {
          name: "網站上的檔案名稱",
          filename: "實際檔案名稱"
        },
        .
        .
        .
      ]
    }

一個縣市請在 data 內開一個資料夾，儲存 .json 的檔案。

有額外可以紀錄的資訊，請自行擴充 json 格式，並且到對應縣市的 issue 內紀錄。
